Spike Pit is a fast-paced, arcade-style volleyball game where strategy meets destruction. Play 1v1 against an AI opponent in a unique 2.5D court where every hit matters and every tile can be destroyed.

This game started as a funny idea I had after playing Nintendo Switch Sports volleyball with my nephews. I thought: "What if every time we spike, the court tiles just... break?" That silly concept stuck with me, and here we are! This is actually my first time making a video game and I used Cursor to code it because I don't know anything about it and I just wanted to quickly test out this idea and see if it's actually fun.

I'd love for you to try it out and let me know what you think! Any feedback, ideas, or suggestions are more than welcome. This is a learning project for me, so your thoughts really help shape where it goes next. :)
